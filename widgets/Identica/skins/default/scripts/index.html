Easy Scroll v1.0 - Unobtrusive content scroller

This is a demonstration of an unobtrusive content scroller. To apply this script you need to

  • Create a div (or any other container element)
  • Assign id to that container (default is "myContent")
  • Adjust it's height in javascript config area
  • Put some content inside the container.
You may also want to style it with css to improve it's appearance.
The content is fully accessible to users with JavaScript disabled browsers.
